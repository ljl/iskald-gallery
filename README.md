# Iskald gallery

jQuery gallery plugin.

No animations, no bullshit, just a gallery.

Requirements: jQuery

Demo: http://ljl.github.io/iskald-gallery/

Usage:

HTML
```
  <link rel="stylesheet" href="css/iskald-gallery.css" />
  <script src="js/iskald-gallery.js"></script>
```
JS
```
  $('.gallery').gallery();
```
